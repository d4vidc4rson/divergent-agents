# Humanize Text — Methodology

Transform sterile AI output into text that reads like a real person wrote it. AI text fails the human test not because it's wrong but because it's too right.

---

## Step 1: Ask for Intensity and Context

Present three levels and ask the user to choose. Do not proceed until they answer. Do not default.

- **Light** -- Professional but warm. Vocabulary cleanup, minor structural loosening.
- **Medium** -- Casual professional. Verbal tics, punctuation quirks, maybe one typo.
- **Heavy** -- 11pm energy. Multiple typos, run-ons, abbreviations, fading effort.

Also ask where it's going (email, LinkedIn, Slack, marketing copy) -- context affects what's appropriate.

---

## Step 2: Apply Transformations

Apply in order, calibrated to selected intensity:

### 2a. Vocabulary Exorcism

**Kill on sight:**
- "delve" -> "dig into" / "look at" / cut
- "dive into" -> "get into" / "check out"
- "leverage" -> "use"
- "utilize" -> "use"
- "navigate" -> "deal with" / "figure out"
- "foster" -> "build" / "help" / cut
- "landscape" -> "space" / "world" / "scene"
- "robust" -> "solid" / "strong" / cut
- "comprehensive" -> "full" / "complete" / cut
- "streamlined" -> "simple" / "clean"
- "game-changer" -> be specific or cut
- "synergy" -> describe what actually happens
- "unpack" -> "break down" / "explain"

**Banned openers:**
- "In today's [X]..." / "In an era of..." -> Start with the actual point
- "Let's be clear..." -> Just be clear
- "It's worth noting..." -> Just note it
- "Interestingly..." -> Let the reader decide
- "Imagine..." / "Picture this:" -> Just start

**Banned structures:**
- "It's not X. It's Y." (faux-profound false dichotomy)
- "X isn't about Y. It's about Z." (same energy)
- "What if I told you..." -> Say the thing

**Kill all glib colon phrases:**
"And here's the kicker:" / "Spoiler:" / "Plot twist:" / "The result?" / "Hot take:" / "Pro tip:" / "The secret?" / "Bottom line:" / "The takeaway:" / "The punchline:"

**Kill dramatic single-sentence paragraphs** and their friends:
"Let that sink in." / "Read that again." / "Full stop." / "Period."

### 2b. Structure Decay

- **Break parallelism** occasionally: "We need speed, we need to be efficient, and honestly just something that works"
- **Add sentence fragments.** For emphasis. Like this.
- **Start sentences with "And" or "But"**
- **Kill the wrap-up bow:** Remove tidy concluding sentences. Cut "In conclusion," / "To summarize," / "Ultimately." End mid-thought occasionally.
- **Interrupt yourself:** Parentheticals, dashes to pivot mid-sentence, trailing thoughts...

### 2c. Punctuation Humanization

- **Reduce em-dashes** -- AI overuses them
- **Inconsistent Oxford comma:** Use sometimes, skip other times
- **Imperfect ellipses:** Two dots.. or four.... occasionally
- **Spacing artifacts:** Occasional double-space after periods

### 2d. Typo and Error Injection (Use Sparingly)

Grammar "mistakes" that pass spellcheck: their/there, its/it's, your/you're, affect/effect, then/than (1-2 max per piece)

Common typos: lowercase "i", "teh", missing words, repeated words, "definately"

Abbreviation mixing: "with"/"w/", "because"/"bc", "tho"/"thru"/"gonna" (context-dependent)

### 2e. Human Verbal Tics

Fillers: "just," "actually," "honestly," "basically," "I mean," "kind of," "sort of"

Softeners: "I think," "I feel like," "maybe," "probably," "not sure but..."

Casual transitions: "Anyway," / "So yeah," / "Oh also--" / "Quick thing:"

### 2f. Energy Inconsistency

- Start strong, get lazier toward the end
- Some sentences polished, others dashed off
- Enthusiasm that fades ("This is great! Really solid. good stuff.")

---

## Intensity Calibration

**Light:** Vocabulary cleanup only. 2-3 sentence fragments. One "just" or "actually." Start one sentence with "And" or "But."

**Medium:** All vocabulary changes. Structure decay. Several verbal tics. 1-2 punctuation quirks. Maybe one lowercase "i."

**Heavy:** Everything above plus multiple typos, run-on sentences, incomplete thoughts, abbreviations, energy decay toward end.

---

## Step 3: Show Before/After

Present the original and transformed version side by side for comparison.

---

## Quality Criteria

- All AI-signature vocabulary is eliminated
- Banned structures and openers are gone
- Transformations match the selected intensity level (not over- or under-applied)
- The result sounds like a specific human, not a different AI style
- Typos and errors (if any) are the kind real humans make, not random character swaps
- The text still communicates the same information clearly
- Energy level and formality match the stated context (email vs Slack vs LinkedIn)

## Self-Check

1. Read the output aloud. Does it sound like a person talking, or a machine trying to sound casual?
2. Are there any surviving AI-signature words or structures?
3. Is the intensity level correct for what the user requested?
4. Does the text still say what it needs to say, or did humanization break the message?
5. Would the target audience (email recipient, LinkedIn connection, Slack channel) find the tone appropriate?
